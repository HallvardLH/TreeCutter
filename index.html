About

Wood Cutter is a JavaScript game. It is still in development, thus many features are unfinished or missing. Some of the art and style is conceptual and will be replaced in the future. The game is best enjoyed in Google Chrome.

If you have any suggestions or find any bugs, feel free to join the Discord server and tell me about them!

Credits

Some of the pixel art in the game has been created by other artist. They let people use their assets for free. Below is a list of all the assets used:
  • Custom cursor hand
  • Battle axe
  • Tree icon used in navigation
  • Hatchet, tomahawk and saw
Thanks to Asgeir for helpful input along the way.
